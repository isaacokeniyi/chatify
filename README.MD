## **CHATIFY**

A global chat application with real time communication

## **ABOUT**

Chatify is a full-stack global chat app for users to send and recieve messages in real-time

## **Live Demo**

[Check out the site](https://chatify-theta-brown.vercel.app/)

## **Features**

- Global messaging
- Real-time communication
- Authentication
- Rate limiting

## **Tech Stack**

- Frontend: React
- Backend: Node.js \&\& Express.js
- Real-time comunication: Socket.IO
- Database: MongoDB
- Styling: Tailwind CSS
- Routing: React Router v7

## **USAGE**

- Sign up / login
- Send and view message from other users
- Right-click on a message to delete
